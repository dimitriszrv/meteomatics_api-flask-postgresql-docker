Project Overview

For this project, we will use only the t_2m:C metric, which represents the instantaneous temperature at 2 meters above ground level in degrees Celsius (C). This parameter will help us achieve the following objectives:

  • List the latest forecast for each location for every day over a 7-day period, with data provided at 3-hour intervals
  • List the average temperature of the last three forecasts for each location for every day
